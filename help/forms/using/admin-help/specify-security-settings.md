# 指定安全設定{#specify-security-settings}

通過輸出，可以控制XML輸入中的外部實體是否被解析。 預設會解決這些問題，但您可以變更此行為以提高AEM表單系統的安全性。

**防止處理包含外部實體引用的XML資料檔案**

1. 在管理控制台中，按一下「服務>輸出」。
1. 清除「解析外部實體」(Resolve External Entities)複選框。
1. 按一下「儲存」。
