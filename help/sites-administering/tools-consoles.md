# 工具控制台{#tools-consoles}

「工 **具** 」主控台可讓您存取許多專業工具，以協助您管理網站、數位資產和內容存放庫的其他方面。 工具控制台目前有兩種 **類型** ，視您使用的UI而定：

* [工具- Classic UI](#tools-classic-ui)
* [工具——觸控最佳化UI](#tools-touch-optimized-ui)
