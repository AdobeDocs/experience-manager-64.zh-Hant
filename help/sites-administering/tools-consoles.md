# 工具主控台{#tools-consoles}

**「工具」**&#x200B;主控台可供存取多種特殊工具，協助您管理網站、數位資產和內容存放庫的其他方面。根據您使用的UI，目前&#x200B;**工具**&#x200B;控制台有兩種類型：

* [工具 — 傳統UI](#tools-classic-ui)
* [工具 — 觸控最佳化UI](#tools-touch-optimized-ui)
